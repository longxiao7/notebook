linux 磁盘扩容

发表于 2019-01-03

记录一下云主机的扩容

假如是设备 /dev/vdb 挂载在 /data 目录

  1. 卸载
  2. 检查磁盘
  3. 调整分区大小
  4. 挂载

上面的情况仅限于 2TB 以内的 MBR 分区。
如果是扩容到 2TB 以上的，从 MBR 转移 GPT 分区，需要先转移文件 然后重新分区。不要在没有备份的情况下轻易尝试直接转换。

参考资料：
https://docs.ucloud.cn/udisk/userguide/extend/linux_raw
